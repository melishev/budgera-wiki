## 1. Добавление счетов

Наличные, карты, вклады, кредиты добавляются в приложение как отдельные счета. Приложение позволяет создать разные счета в разных валютах. В приложении, по умолчанию, уже созданы счета "Наличные" и "Карта".

Остальные счета можно добавить тремя способами:

1. **Из интернет-банка**, подключив интеграцию с поддерживаемым банком. В дальнейшем операции по этим счетам будут автоматически импортироваться из банка. Чтобы добавить счета, перейдите на вкладку Счета, нажмите (+) в правом верхнем углу и выберете нужный вам банк.
2. **Из банковских SMS.**&#x20;[**\* только для Андроид**] В дальнейшем операции по этим счетам будут добавляться автоматически при получении SMS от банка. Чтобы добавить счета, перейдите в Ещё / Журнал SMS, нажмите “Распознать SMS” и выберите период, за который добавить операции.
3. **Вручную.** Операции по этим счетам надо будет вносить вручную. Чтобы добавить счета, перейдите в раздел Счета, нажмите (+) в правом верхнем углу и “Добавить счёт без подключения”.

Чтобы упростить дальнейший учёт операций и уже на первом этапе получить данные для анализа, мы рекомендуем добавить сразу счета и операции из интернет-банка из SMS.

::warning
Все подключения к банкам осуществляются только в мобильном приложении! Автоматическая загрузка операций из интернет-банка или СМС происходит через мобильное приложение. На сайте доступно только редактирование уже загруженных операций или ручное добавление операций.
::

## 2. Укажите, сколько у вас денег на счетах

При регистрации в приложении были созданы счета "Наличные" и "Карта" с нулевыми балансами. Укажите, сколько у вас денег на этих счетах. При необходимости скорректируйте остатки и на других счетах.

Чтобы скорректировать остатки, перейдите внутрь нужного счёта, нажмите цифры остатка на счете – откроется окно редактирования этого остатка, где нужно указать правильный ТЕКУЩИЙ остаток. Для первоначального редактирования остатка после добавления счета рекомендуется выбрать пункт «Сохранить без добавления операции».

## 3. Добавьте планируемые операции

Добавьте будущие планируемые операции, например, с поступлением зарплаты или ваши ежемесячные платежи: коммунальные, телефон, интернет, и др. На основе этой информации будет построен календарь ваших расходов и настроены Планы.

Чтобы добавить планируемую операцию, перейдите на вкладке «Операции», нажмите красный (+) внизу экрана, выберите тип операции, в форме добавления операции укажите будущую дату ближайшего платежа и периодичность повторения.

## 4. Добавьте дополнительный логин

Добавьте дополнительный логин для вашей супруги или супруга, чтобы вести семейный учёт.

Семейный учёт даёт возможность пользоваться одним аккаунтом нескольким членам семьи. У каждого пользователя будет свой логин в систему, свои личные счета и свои настройки для комфортной работы с приложением. Подробнее про совместный учёт читайте в Справке.

Чтобы добавить дополнительный логин, перейдите на вкладку Ещё / Совместный учёт / Добавить нового пользователя.
